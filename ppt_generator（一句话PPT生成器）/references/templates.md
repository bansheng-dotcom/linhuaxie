# 10套行业模板配色与版式规范

## 1. 专业商务风（职场汇报/年终总结）
- 主色：#1A3A5C（深蓝） / 辅色：#FFFFFF（白） / 强调色：#D4AF37（金）
- 字体：标题思源黑体 Bold 28pt / 正文微软雅黑 18pt
- 版式：左对齐，大标题栏深蓝底色+白字，内容区白底+蓝字
- 图表：柱状图/折线图主色深蓝，关键数据用金色标注

## 2. 极简科技风（创业路演/融资BP）
- 主色：#FFFFFF（白） / 辅色：#000000（黑） / 强调色：#0071E3（苹果蓝）
- 字体：标题SF Pro Display 32pt / 正文SF Pro Text 18pt
- 版式：居中排版，大留白，每页最多5行字
- 图表：深色背景+浅色数据条，关键数字放大3倍

## 3. 营销视觉风（产品发布会）
- 主色：品牌色 / 辅色：深灰 / 强调色：品牌亮色渐变
- 字体：标题大黑粗体 36pt / 正文圆体 16pt
- 版式：全屏大图背景+半透明蒙层+标题居中
- 图表：大字号产品卖点，少图表、多视觉冲击

## 4. 学术清新风（教育培训）
- 主色：#3B82F6（浅蓝） / 辅色：#FFFFFF（白） / 强调色：#F97316（橙）
- 字体：标题微软雅黑 Bold 28pt / 正文宋体 18pt
- 版式：顶部章节条+内容区留白充足，左侧可加目录
- 图表：浅色底+蓝色数据条+橙色高亮

## 5. 数据图表风（数据分析报告）
- 主色：#1F2937（深灰） / 辅色：#3B82F6（蓝） / 强调色：#10B981（绿）
- 字体：标题Roboto Bold 24pt / 正文Roboto 14pt
- 版式：左侧文字说明+右侧大图表，比例3:7
- 图表：配色严格遵循数据可视化规范，红=降/绿=升

## 6. 逻辑架构风（方案提案）
- 主色：#1E3A5F（深蓝） / 辅色：#E2E8F0（浅灰） / 强调色：#F59E0B（高亮黄）
- 字体：标题微软雅黑 Bold 26pt / 正文微软雅黑 16pt
- 版式：流程图/架构图为主，文字作为图例说明
- 图表：大量使用箭头/连线/层级图

## 7. 创意设计风（个人简历/作品集）
- 主色：自定义（用户可选） / 辅色：#FAFAFA / 强调色：用户主色加深
- 字体：标题艺术字 36pt / 正文无衬线 14pt
- 版式：左右分栏（3:7），左侧个人信息+右侧作品展示
- 图表：作品缩略图为主，少文字

## 8. 活力暖色风（活动策划）
- 主色：#F97316（橙） / 辅色：#EF4444（红） / 强调色：#FFFFFF（白）
- 字体：标题大黑粗体 32pt / 正文黑体 16pt
- 版式：全屏暖色渐变背景+白色文字，活泼排版
- 图表：icon为主，少传统图表

## 9. 简洁高效风（周报/月报）
- 主色：#FFFFFF（白） / 辅色：#3B82F6（品牌蓝） / 强调色：图表色
- 字体：标题微软雅黑 Bold 24pt / 正文微软雅黑 14pt
- 版式：上下结构，页面上方标题+下方2-3列卡片
- 图表：KPI卡片式布局，关键数字大号加粗

## 10. 学术严谨风（论文答辩）
- 主色：#FFFFFF（白） / 辅色：#1E3A5F（藏蓝） / 强调色：#6B7280（灰）
- 字体：标题宋体 Bold 24pt / 正文宋体 16pt
- 版式：顶部校徽+标题，底部页码+院系名
- 图表：学术规范配色，表格简洁无背景色
