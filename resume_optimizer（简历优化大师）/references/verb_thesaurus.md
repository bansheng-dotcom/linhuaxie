# 弱动词 → 强动词 替换表

## 通用类

| 弱动词 | 强动词 | 适用场景 |
|--------|--------|------|
| 负责 | 主导 / 搭建 / 驱动 / 统筹 / 带领 | 有主导权的工作 |
| 参与 | 推动 / 协同 / 贡献 / 加入核心组 | 需要指明具体贡献 |
| 做 | 设计 / 开发 / 实施 / 交付 / 创建 | 有产出物的任务 |
| 帮忙 | 支持 / 协助 / 配合 | 需指明成果 |
| 沟通 | 跨部门协调 / 向上汇报 / 对外谈判 / 对接 | 区分沟通对象 |
| 学习 | 掌握 / 快速应用 / 自研 / 上手 | 体现学习成果 |
| 处理 | 解决 / 化解 / 优化 / 消除 | 强调结果 |
| 维护 | 保障 / 守护 / 确保稳定 | 适用于SRE/SLA场景 |
| 写 | 撰写 / 产出 / 输出 / 编写 | 不同类型文档 |
| 看 | 审查 / 审核 / 评估 / 分析 | 不同审阅场景 |
| 说 | 汇报 / 分享 / 培训 / 路演 | 区分场合 |
| 想 | 策划 / 设计 / 构思 / 规划 | 体现思考深度 |

## 严禁使用的词

以下词汇应**全量替换**，不加任何例外：
- "负责" → 这是最弱的词，必须换
- "参与" → 除非是国家级/行业级重大项目
- "各种"、"一些"、"等等" → 模糊词，全删
- "大概"、"差不多" → 不专业
