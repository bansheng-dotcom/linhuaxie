# 各类型合同常见陷阱模式库

## 劳动合同
- **试用期过长**：合同期1年试用期超2个月、合同期3年试用期超6个月 → 违反《劳动合同法》第19条
- **竞业限制范围过大**：竞业限制超过2年、覆盖全行业 → 第24条
- **违约金过高**：除培训费+竞业限制外不得约定违约金 → 第22/25条
- **工资构成模糊**：只写"按公司薪酬制度执行" → 应明确基本工资/绩效/补贴
- **单方变更权**：保留"公司有权调整岗位/薪资"的文字 → 属于变相侵害劳动者权益
- **加班费规避**：约定"月薪含所有加班费" → 违反劳动法加班费计算标准
- **合同期限-试用期**：只签试用期合同不签正式合同 → 试用期包含在合同期内

## 租房合同
- **押金不退陷阱**：条款模糊如"房屋有损坏则不退押金" → 应明确何为"损坏"
- **维修责任转嫁**：约定"租客承担所有维修费" → 法定义务在房东（非人为损耗）
- **单方涨租权**："房东有权根据市场行情调整租金" → 租金应固定或约定调整机制
- **提前解约惩罚过重**：违约金超3个月房租 → 违约金应与实际损失相当
- **转租限制**：禁止转租但无例外 → 建议增加"经房东书面同意可转租"
- **续租条件不明**：未约定续租流程和提前通知期限
- **水电费分摊不清**：尤其是合租房，分表计量方式未约定

## 服务/外包合同
- **验收标准模糊**：只说"达到甲方满意" → 需约定客观可量化的验收标准
- **付款节点不合理**：全部款项在验收后支付 → 应分阶段付款
- **知识产权归属**：约定"所有产出物归甲方"但单价极低 → IP对价与单价不匹配
- **违约金不对等**：乙方违约金远高于甲方 → 显失公平
- **无限连带责任**：约定乙方承担甲方全部间接损失 → 建议限定额度
- **排他性条款**：禁止乙方为任何其他客户服务 → 影响生计

## 买卖合同
- **质量标准引用模糊**：写"符合国家质量标准" → 应注明具体标准编号
- **交付风险转移**：约定"发货后风险由买方承担" → 可能转嫁运输风险
- **验收期过短**：电子产品验收期只给24小时 → 不合理
- **争议管辖地**：约定在卖方所在地法院管辖 → 增加买方维权成本
- **质保期起算**：从"发货日"而非"到货日"起算 → 实际质保期缩短

## 投资/合伙协议
- **退出机制缺失**：没有约定怎么退出、按什么估值退出
- **决策权集中**：一方拥有"一票否决权"且无制衡机制
- **反稀释条款**：无条件反稀释 → 对创始团队极度不利
- **清算优先权**优先级过高 → 普通合伙人可能清零
- **竞业禁止过广**：限制范围涵盖不相关领域
- **代持风险**：股权代持协议法律效力不完整

## 保密协议(NDA)
- **保密范围过宽**：定义为"甲方披露的所有信息" → 应限定为"明确标注为保密的信息"
- **保密期限永久**：约定"保密义务永久有效" → 合理期限3-5年
- **违约金过高**：约定"违约金为合同金额的10倍" → 应与实际损失关联
- **单方保密**：只约束一方 → 合理的NDA应双向

## 民间借贷
- **利率超标**：年利率超过LPR的4倍 → 超额部分法律不支持
- **砍头息**：预先从本金扣除利息 → 违法
- **担保约定不明**：未明确是一般保证还是连带保证
- **利滚利（复利）**：约定未还利息计入本金再计息 → 需看是否超过法定上限
