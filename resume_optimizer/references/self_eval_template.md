# 自我评价改写公式与模板

## 黄金公式

```
3句话 = [行业年限/学历背景] + [核心能力×3] + [最高成就数据] + [求职意向]
```

## 模板1：有经验的职场人

**【公式】**
[X]年[行业/领域]经验，擅长[能力1]、[能力2]和[能力3]。
主导过[最有代表性项目]，实现[量化成果]。
寻求[岗位方向]机会，期待[价值主张]。

**【示例】互联网产品经理**
5年B端SaaS产品经验，擅长需求分析和产品规划。
主导的核心产品线年度ARR从1200万增长至3000万（+150%），客户续约率92%。
寻求可带团队的产品负责人岗位，期待在AI应用方向深耕。

**【示例】后端开发工程师**
7年Java后端开发经验，精通高并发系统设计和微服务架构。
主导的订单系统重构项目支撑日均交易量从50万笔增长至300万笔，系统可用性从99.9%提升至99.99%。
寻求技术专家/架构师方向机会，T型发展。

## 模板2：1-3年经验

**【公式】**
[X]年[领域]实操经验，独立完成过[代表性项目]，沉淀了[核心能力]。
擅长[差异化能力1]和[差异化能力2]。
目前聚焦[方向]，期望在[目标岗位]继续深耕。

**【示例】运营岗**
3年电商运营经验，独立负责过年GMV 2000万+店铺的全盘运营，沉淀了一套数据驱动的精细化运营方法论。
擅长付费投流和私域社群裂变，将店铺ROI从1.8提升至3.5。
目前聚焦品牌电商赛道，期望在增长运营岗位继续深耕。

## 模板3：应届毕业生

**【公式】**
[学历背景]，[实习经历亮点]，[掌握的硬技能]。
在校期间[代表性项目/竞赛/论文]。
[性格特质]，对[目标行业]充满热情。意向岗位：[具体岗位]。

**【示例】计算机专业**
985计算机本硕，GPA 3.8/4.0，3段互联网大厂实习经历（字节/腾讯/阿里）。
主导的毕业设计获院级优秀论文（Top 2%），LeetCode刷题400+，熟悉Java/Go/Python技术栈。
逻辑清晰、自驱力强，对分布式系统方向充满热情。意向岗位：后端开发工程师。

**【示例】商科**
211市场营销本科，TOP快消公司暑期实习经历，独立策划的校园推广方案获华东区第二名。
熟练掌握SPSS/Python数据分析、Tableau可视化。
沟通能力强（曾任学生会外联部部长），对消费品赛道有深入认知。意向岗位：品牌管培生。

## 模板4：转行求职者

**【公式】**
[原行业年限]经验转为[新领域方向]，[原行业可迁移的核心能力]与新岗位高度契合。
自学[新技能]并完成[实践项目/作品]，证明了快速学习能力。
致力于将[原行业经验]与[新领域]结合，创造独特价值。

**【示例】土木→数据分析**
5年土木工程经验转为数据分析方向，积累的严谨数据处理习惯和跨部门协作能力与数据岗位高度契合。
自学SQL/Python/Tableau并完成3个独立分析项目（电商用户画像、餐饮选址模型、房价预测），Kaggle比赛Top 5%。
致力于将工程领域的数据思维与商业分析结合。

## 自我评价禁忌

❌ "本人性格开朗、吃苦耐劳、认真负责" → 空话，无区分度
❌ "学习能力强、适应能力好" → 所有人都会写，等于没说
❌ "希望得到一个学习成长的机会" → 公司不是学校
❌ 超过5句话 → 自我评价最长3句话
❌ 第一人称"我" → 简历通常省略主语
